# UI/UX Design for SOFTIT App

This repository contains the UI/UX design and code bundle for the **SOFTIT Feedback Application**.  
It includes the Kiosk APK interface for tablets and the Admin & Client dashboards for web.  

## Figma Preview

You can view the interactive design preview here:  
![SOFTIT App Preview](/1feedback.png)

This preview shows:  
- Kiosk interface for tablet feedback input  
- Admin dashboard for managing templates, users, reports, and settings  
- Client dashboard to customize feedback questions, templates, colors, logo, and view reports  

---

## Project Structure

- `src/` → Contains all the React/TypeScript source code for the project  
- `index.html` → Entry point of the web project  
- `package.json` → Project dependencies and scripts  
- `vite.config.ts` → Vite configuration for development  
- `README.md` → Project documentation

---

## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
